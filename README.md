# HSK Daily Rush 🇨🇳

**A Mobile-First, Immersive, Gamified EdTech App for HSK Practice.**

HSK Daily Rush is an interactive learning platform designed to help students master Mandarin Chinese (HSK syllabus). Inspired by popular language apps, it features engaging game mechanics, bouncy 3D UI, and real-time feedback for a seamless learning experience.

## ✨ Features

- **Gamified Learning:** Pinyin drag & drop, vocabulary matching, and manual text input.
- **Mobile-First Design:** Optimized for mobile screens with bubbly, tactile UI elements.
- **Interactive Mechanics:** Smooth animations, haptic feedback, and dynamic audio cues.
- **Progress Tracking:** Hearts system, streaks, scores, and daily hints.
- **Offline Capable:** Local storage persistence for game progress and high scores.

## 🚀 Technology Stack

- **Framework:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **State Management:** Zustand
- **Audio & Visuals:** Web Audio API, HTML5 Canvas, SVG

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/praprastyo/hsk-daily-rush.git
   cd hsk-daily-rush
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:5173`).

### Building for Production

To create a production-ready build:
```bash
npm run build
```

To preview the built app:
```bash
npm run preview
```
